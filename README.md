# Bank Loan Management System

## Description
This is a Java-based Loan Management System designed to manage different types of loans such as:
- Personal Loan
- Home Loan
- Business Loan
- Car Loan
- Student Loan

## Features
- Loan creation using Factory Pattern
- Input validation
- Loan repayment tracking
- OOP concepts (Encapsulation,Interface,Inheritance,Polymorphism,Constructors,Method overriding )

## Technologies
- Java
- OOP Principles

## How to Run
1. Open project in IntelliJ
2. Run `Main.java`
